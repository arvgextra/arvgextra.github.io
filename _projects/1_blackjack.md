Developing a real-time computer vision system using **YOLOv8** and **ByteTrack** to detect and track playing cards, integrating a **Hi-Lo card counting engine**, a finite state machine for live strategic recommendations, and a reinforcement learning agent to optimize betting and play decisions.

**Stack:** YOLOv8 · ByteTrack · PPO (RL) · Hi-Lo · Python · OpenCV
