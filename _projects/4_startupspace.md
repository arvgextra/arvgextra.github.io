Group project building a **startup-investor matchmaking platform** with four distinct user personas: founders, venture capitalists, acquirers, and general researchers. The platform lets startups publish financial metrics and documents, VCs track deal flow, acquirers manage acquisition pipelines, and researchers publish and comment on insights.

Backend is a normalized **MySQL relational database** with 15+ tables covering startups, founders, investors, documents, financial metrics, and deal tracking. Exposed via a **Flask REST API** with role-specific routes, and fronted with an Appsmith dashboard. Includes a Jupyter notebook for term statistics analysis.

**Stack:** MySQL · Flask · Python · Docker · Appsmith · Jupyter
