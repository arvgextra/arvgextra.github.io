Built a book recommendation system using Goodreads data, combining **sentiment analysis** on review text with **KMeans clustering** across multiple book attributes. Recommendations are generated by computing centroid distance similarity between a user-input book and other titles within the same cluster. Results are surfaced through a dashboard for easy exploration.

**Stack:** Python · KMeans · Sentiment Analysis · Web Scraping · Goodreads data
