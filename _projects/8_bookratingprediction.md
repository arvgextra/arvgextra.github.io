Explored whether book descriptions, textual characteristics, and author history can predict reader ratings before reviews exist. Extracted TF-IDF and embedding features from descriptions alongside metadata (genre, publication year, page count, author rating history) and trained two models: a **Neural Network** in Python and a **Bayesian Regression Model** in R. Compared models using description features alone vs. full metadata to evaluate the predictive value of language vs. context.

**Stack:** Python · PyTorch · R · TF-IDF · Bayesian Regression
